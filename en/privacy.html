Barre Application - Task Manager

Version 1.0 | Date: 2024

📊 Mental Load & Thread System

Barrë uses a "thread" system to help users visualize and manage their mental load. A thread represents:

  • A task, appointment, or thought on user's mind
  • Associated category (Home, Family, Work, Health, Other)
  • Status (Active or Half-Done)
  • Creation date

All threads are stored ONLY locally on the user's device. No threads are sent to external servers or shared without explicit user action (Share feature).

🔄 Share Feature

Users can voluntarily share their mental load summary. This feature:

  • ✅ Generates a text summary of threads
  • ✅ User controls what to share
  • ✅ Shared via standard share dialog (Messages, Email, etc.)
  • ✅ Does NOT store shared data on servers
